# Vue2开发的音乐播放器

技术栈: Webapck / ES6 + Babel / Vue2 / Vue Router / axios / vuex

> 音乐播放器

## 安装
```shell
$ git clone https://github.com/lijiliang/vue-music
$ cd vue-music
$ npm install
```

## 开发模式
```shell
npm start
```
无意外，默认浏览器就会自动打开 `localhost:8080`，您立即可以看到效果

若浏览器没有自动弹出，则请自行手动访问  

## 构建模式
```shell
npm run build
```
